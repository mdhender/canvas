Basic List of Posts

I want my home page to show a list of my posts with the newest posts listed first. The layout for this uses sections to wrap the summary: <section> <header> <h2>Post Title</h2> </header> <p>Post Summary</p> <nav> <ul> <li><a href="Post Link">Read More &raquo;</a></li> </ul> </nav> </section> For each summary, we need to Replace “Post Title” with the real title. Replace “Post Summary” with a summary of the post. Replace “Post Link” with a link to the full post.

I want to be able to do something like: $ hugo new posts/creating-a-post-archetype.md From the previous post, we know that Hugo will look for archetypes/posts.md. First Look at Archetypes

This post is about archetype files. Probably too much information, but it all starts with the archetype. What’s an Archetype file? Archetypes live in the archetypes/ directory in the root of your site.
